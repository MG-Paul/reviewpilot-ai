# ReviewPilot AI

An AI-powered Systematic Review Copilot for biomedical researchers. Designed to automate workflows (import, duplicate detection, title/abstract screening, grounded extraction, Risk-of-Bias audits, and reactive meta-analyses) while maintaining human-in-the-loop validation.

## Scaffolding Structure
* `/frontend`: Next.js web application.
* `/backend`: FastAPI Python backend.
* `/r-sandbox`: Dockerized R execution sandbox.
